# salami
salami is a python package for volume electron microscopy

(Slice And Look At My Insides)



## Getting Started

1. Install OpenFIBSEM (https://github.com/DeMarcoLab/fibsem)

2. Clone this repo.

3. Install Salami
```
$ cd salami
$ pip install -e.
```

4. Run viewer
```
$ salami_viewer
```
